fn auth_weight( auths: impl IntoIterator<Item = impl Borrow<Self>>, db_weights: RuntimeDbWeight ) -> Weight

Computes weight of the given SignatureWithNonce. Considers the no. and types of signatures and no. of reads. Disregards message size as messages are hashed giving the same output size and hashing itself is very cheap. The extrinsic using it might decide to consider adding some weight proportional to the message size.
